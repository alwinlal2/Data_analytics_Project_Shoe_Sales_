# Data_analytics_Project_Shoe_Sales_
End-to-end sales analytics project using Excel, MySQL, and Power BI with interactive dashboards and business insights.

Footwear Sales Analysis

Project Overview

This project demonstrates an end-to-end data analytics workflow using Excel, MySQL, and Power BI. 
The objective is to analyze footwear sales data, uncover business insights, and build interactive dashboards for decision-making.

---

**Tools Used**

- Microsoft Excel
  - Data Cleaning
  - Helper Columns
  - Pivot Tables
  - Dashboard Design

- MySQL
  - Data Quality Checks
  - Business Analysis Queries
  - CTEs
  - Window Functions
  - Ranking Functions

- Power BI
  - Interactive Dashboard
  - KPI Cards
  - Slicers
  - Dynamic Visualizations


**Key Business Questions**

- Which brands generate the highest revenue?
- Which shoe types sell the most units?
- What are the monthly and quarterly revenue trends?
- Which sales channels perform best?
- Which countries contribute the highest revenue?
- How do different price bands perform?
- What are the top-performing markets and products?

**Dashboard Features**

- Executive KPI Summary
- Monthly Revenue Trend
- Revenue by Country
- Revenue by Sales Channel
- Revenue by Price Band
- Interactive Filters for:
  - Month
  - Brand
  - Country
  - Shoe Type
  - Sales Channel
